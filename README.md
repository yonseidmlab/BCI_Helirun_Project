# BCI_Helirun_Project

전반적인 설치 가이드라인에 대해선 아래의 링크를 참조하세요. 
https://docs.google.com/document/d/1uWdjW5s1uQayeMgwd3Y5VtT6Q42U3DMjzesE2Jti14Y/edit#
